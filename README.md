# Secretary(vaporware)

> Note:Japanese only yet.

TODO,予定,の組み立て・管理を行うプログラムです。自分用。

## Features

- TODO list
- Scheduler
  - TODO list にある要素をtaskとしてcalendarに登録する。
  - Manage time tables by a minute
- Recorder
  - Userが自信の実際の行動を記録できる
  - Schedulerと比較して、何ができて何ができなかったかを自動で評価し、統計をとる
